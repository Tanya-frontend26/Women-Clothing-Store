# 👗 Women Clothing Store

A responsive Women Clothing Store website built using **HTML, CSS, and JavaScript**. This project showcases a modern fashion store interface with clean UI, responsive design, and interactive JavaScript functionality.

---

## ✨ Features

* Responsive design for desktop and mobile
* Modern and clean user interface
* Product showcase section
* Navigation bar
* Interactive JavaScript functionality
* Easy-to-use layout

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
